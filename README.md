# PythonWordle
Wordle, coded in python!

Wordle is a fun game in which the user tries to guess a word! 

HOW TO PLAY
The user starts by entering a 5 letter word. They are trying to guess a 5 letter word. 
If a letter is shared by the 2 words, it shows up yellow. 
If the shared letter is in the right position, it shows up green. 

Using yellow and green letters, can you guess the word in 6 tries?
